### Código de implementação da Máquina de Turinng

#### Foi definida uma máquina de Turing, que complementa uma entrada binária na fita, ou seja, uma entrada "1100011", por exemplo, será transformada em "0011100".

1. Σ = {0, 1}
2. Q = {init, final}
3. q 0 = inicial
4. q f = final

| para o certo. |
